## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/alaa-alhajj/trello-todo-app.git
cd trello-todo-app
```

### 2. Install dependencies
```bash
npm install
```


### 3. Run the development server
```bash
npm run dev
```
Open http://localhost:3000 in your browser.
